# TypeTheScript - Simple Monkeytype Clone

## Getting started

1. Clone this repository

    ```
    git clone https://github.com/dwisarut/type_the_script.git
    cd type_the_script
    ```
2. Install dependencies

    Installing all depedencies for frontend, then install it in backend
    
    ```
    npm install
    ```
3. Run this project
    ```
    npm run dev
    ```
